# SuperStore Sales Data Analysis Dashboard

## Project Overview
This repository contains an interactive Data Analytics dashboard built to analyze sales performance and customer trends using the **SuperStore Sales Dataset**. The primary goal of this project is to transform raw sales data into meaningful, actionable insights that can drive strategic business and marketing decisions.

## Key Objectives
* **Trend Analysis:** Analyze overall sales and customer acquisition trends over time.
* **Product Performance:** Identify top-performing products, categories, and seasonal patterns to optimize inventory and marketing efforts.
* **Strategic Insights:** Derive data-backed insights to support overarching business strategies.

## Tools & Technologies Used
* **Business Intelligence:** Microsoft Power BI
* **Data Source:** CSV (`SuperStore_Sales_Dataset.csv`)
* **Techniques:** Data Cleaning, Data Modeling, Data Visualization (Bar Charts, Line Charts, KPIs)

## Repository Contents
This repository contains the raw dataset and the internal structural files of the Power BI project:
* `SuperStore_Sales_Dataset.csv`: The raw dataset containing detailed transactions, product details, and customer information.
* **Power BI Internal Files:** `DataModel`, `DiagramLayout`, `Metadata`, `Settings`, `SecurityBindings`, `Version`, and `[Content_Types].xml`.xml]. These represent the unzipped underlying data and schema of the Power BI report. 

> **Note to Viewers:** To fully interact with the dashboard, download the repository and open the associated `.pbix` file (if uploaded) in Power BI Desktop.

## Key Visualizations
* **KPI Cards:** High-level metrics providing instant visibility into Total Sales, Total Profit, and Total Orders.
* **Line Charts:** Highlighting seasonal patterns and month-over-month sales velocity.
* **Bar Charts:** Ranking top-performing products, regions, and customer segments.
